# 快速开始指南

## 第一步：安装依赖

```bash
npm install
```

## 第二步：配置环境变量

**重要**：你需要手动创建 `.env` 文件。

在项目根目录创建 `.env` 文件，内容如下：

```
AI_BUILDER_TOKEN=sk_7b71f59a_3fda86b315eb6ba3f8aaad4be2ee5d8998c2
PORT=3000
```

**注意**：
- 请确保 `.env` 文件在 `.gitignore` 中（已自动配置）
- 不要将 token 提交到代码仓库

## 第三步：启动服务器

```bash
npm start
```

或者使用开发模式（自动重启）：

```bash
npm run dev
```

## 第四步：打开浏览器

访问：`http://localhost:3000`

## 使用提示

1. **首次使用**：浏览器会请求麦克风权限，请点击"允许"
2. **开始录音**：点击"开始录音"按钮，说完后点击"停止录音"
3. **查看对话**：系统会自动处理并显示对话内容
4. **继续对话**：可以继续点击"开始录音"进行下一轮对话

## 故障排除

### 无法访问麦克风
- 检查浏览器权限设置
- 确保使用 HTTPS 或 localhost（HTTP localhost 也可以）

### API 调用失败
- 检查 `.env` 文件中的 `AI_BUILDER_TOKEN` 是否正确
- 检查网络连接
- 查看服务器控制台的错误信息

### 录音没有声音
- 检查麦克风是否正常工作
- 检查系统音量设置
- 尝试刷新页面重新授权
